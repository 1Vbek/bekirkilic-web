## 5 · Animasyon Rehberi

| Element | Animasyon |
|---|---|
| Bölüm başlıkları | fade-up 0.8s cubic-bezier(.22,.61,.36,1), IntersectionObserver ile tetiklenir |
| Kart gridler | Staggered (0.1s / 0.2s / 0.3s gecikme) |
| Hero görseli | Scale-in + hafif scroll parallax (0.05 oranı) |
| Hizmet kartları | Hover'da translateY(-8px) + gradient doldurma + ikon dönüş |
| Neden biz kartları | Hover'da sağa 6px kayma + border renk değişimi |
| Floating rozetleri | 6s ease-in-out infinite (yukarı-aşağı) |
| Açık göstergesi | 2s pulse animasyonu |
| Marquee | 40s linear infinite scroll |
| Menü açma | slideDown 0.3s |

Reduced-motion tercihinde tüm animasyonlar devre dışı — erişilebilirlik ilk sırada.
